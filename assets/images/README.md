# 图片资源目录

这个目录用于存放网站的图片资源，包括：

## 必需文件
- `avatar.jpg` - 用户头像（建议尺寸：100x100px）
- `favicon.ico` - 网站图标（建议尺寸：16x16px, 32x32px）
- `apple-touch-icon.png` - iOS设备图标（建议尺寸：180x180px）

## 可选文件
- `logo.png` - 网站Logo
- `banner.jpg` - 首页横幅图片
- `default-post.jpg` - 文章默认封面图片

## 图片优化建议
- 使用适当的图片格式（JPG用于照片，PNG用于图标和透明图片）
- 压缩图片大小以提高加载速度
- 考虑使用WebP格式以获得更好的压缩率
- 为不同设备提供不同尺寸的图片
